# Public Storage Facility (Java, MySQL, JavaFX)

## Project Overview
This project is a database-driven system to manage rooms, customers, and inventory for a storage facility. I personally contributed to the database–GUI integration using MySQL and JavaFX in Visual Studio Code.

## My Contributions
- Implemented database–GUI integration to handle room, customer, and inventory data.
- Coordinated team responsibilities across front-end, back-end, and database development.
- Refactored and improved code for better readability and maintainability.

## Features
- Add, update, and remove storage rooms and customers.
- Track inventory for each storage unit.
- Simple GUI for interaction with the database.
- Example data included for testing.

## How to Run
1. Make sure you have Java, MySQL, and JavaFX installed.
2. Update database connection details in the project configuration.
3. Compile and run the project using Visual Studio Code or any Java IDE.

- Java
